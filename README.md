# Tele-Drive
Tele-drive adalah project pribadi, sebagai jembatan sederhana untuk menjadikan telegram sebagai cloud, layaknya seperti google drive.

cara penggunaan buat chatbot ditelegram, lalu set konfigurasi pada halaman dashboard, isikan disana nomer api bot, dan chatid bot kalian lalu simpan.
setelah itu teledrive sudah bisa digunakan untuk menguploud data kalian, dan sekarang bot kalian sudah jadi drive pribadi.
website ini masih dalam tahap pengembangan,jika masih ada yang error/fitur terbatas, mohon dimaklumkan.
